# 🏏 IPL Dashboard Project

## 📌 Project Overview

This project focuses on analyzing Indian Premier League (IPL) data and presenting insights through an interactive dashboard built in Excel.

The dashboard helps in understanding team performance, player statistics, and match trends.

---

## 🎯 Objectives

* Analyze IPL match data
* Identify top-performing teams and players
* Track season-wise trends
* Provide business insights using data visualization

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Data Cleaning Techniques

---

## 📊 Key Insights

* 🥇 Top-performing teams based on wins
* 🏏 Highest run scorers and best players
* 📈 Match trends across seasons
* ⚡ Toss impact on match results
